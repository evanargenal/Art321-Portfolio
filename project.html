Art 321 Evan Argenal - Final Project


Turtle Chase


For my final project, I created a simple game in Python using Turtle graphics.
The goal of the game is to catch the orange turtle which can be achieved by getting at least within 30 units of eachother.
I was able to embed the Python code/game into my website using trinket.io

Controls:
"w" = Move Up
"a" = Move Left
"d" = Move Right
"s" = Move Down
